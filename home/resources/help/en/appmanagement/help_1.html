Operation Steps
Make selection by using the direction keys of TVpad remote,
and pressto enter.

Function
TVpad Store , Installed Apps, Others.

How to use the functions of TVpad Store
·Search: Select the search bar, and pressto input
            keywords you want with soft keyboards by the
            remote. Pressand select “Search” to view results.
(Remarks:Supporting input method of multi-languages, press
 <Input Method> button on the TVpad remote to switch.)
·More：Select “More” and pressto display the list of all
           apps for updating in order of date.
·Category: Select “Category” and pressto display the app
               categories and then select a specific category to
               view related apps.
