Details about the Function Keys
1.App Manager: All functions include TVpad Store,
                           (download), Installed apps (upgrade
                           or uninstall), and Others (install).
2.My TVpad:Recharge, Recharge Records, Messages,
                      Activate Warranty.
3.Settings: System setting list includes Network Test,
                    Network Settings (wired network), Wi-Fi
                    Settings (wireless network), Language
                    Settings, Time Zone Settings, Video
                    Settings, Formatting, System Upgrade,
                    Factory Default, System Info, and About.
