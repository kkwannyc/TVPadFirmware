Operation Steps
1.Insert the external storage devices (including U disks,
  TFcard, SDcard or mobile hard disk etc.) into TVpad,
  and wait for the status bar to show the corresponding
  icon, which indicates the external storage device is identified
  successfully.
2.Use the direction keys to select the icon of external devices,
  and then press.
3.In the pop-up dialogue box, use the direction keys to select
  “OK”, and then press.
4.Wait for the system to format, and pressto exit the
   program after the system prompts “Formatting succeeded”.

Description
Format an external USB device into EXT3 format to meet
the normal play of apps (supporting the external device of
a single partition only, and make sure that your USB is not
set with subfolders). After formatting the default format of
device system file is EXT3.  EXT3 is a kind of journal file
