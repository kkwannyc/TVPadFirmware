Operation Steps
Use the TVpad remote to control the following settings and
pressto enter.

List of Settings:
Network Test       Detect the status of the current
                             network connection; if it is
                             abnormal, check and reset the
                             network connection.
Network Settings   Configure your TVpad wired network
                             with the correct data.
Wi-Fi Settings       Configure your TVpad wireless network
                             with the correct data.
Language Settings TVpad supports multi-language system
                             environment.
Time Zone Settings Set time zone for TVpad.
Video Settings      Switch the video output mode and TV
                             system.
Formatting           Format the external storage device.
