Operation Steps
1.Select “Test It Now” and pressbutton on TVpad remote,
   and the system begins to detect the network connection.
2.If the result is “Normal”, press to exit the program.
3.If the result is “Abnormal”, reset the network in “Network
  Settings” or “Wi-Fi Settings” after exiting.

Description
Detect current network connection of this device including IP
address, gateway, DNS, and server, etc. If it is abnormal,
please check and reset the network.
