Operation Steps
1.Insert the external storage device（including USB disk,
   TF card, SD card, mobile hard disk and etc.）with latest
   upgrade files into TVpad, wait until the icon is displayed
   in the status bar, which means the external storage device
   is identified (online installation can skip this step).
2.Enter the system upgrade page, use the direction keys to
   select “ok”, and then press.
3.Wait for TVpad to restart after the system prompts “The
   system upgrade succeeded”.
4.The system upgrad succeeds after restarting.

Upgrade Methods
·Offline upgrade: Copy the upgrade files into the storage
                       device, insert it into the USB port of
                       TVpad, and then choose “System Settings
                       >System Upgrade>OK “.
·Online Upgrade: Connect to the network, and choose
                        “System Settings >System Upgrade > OK”.
