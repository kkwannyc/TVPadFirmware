Operation Steps
Automatically get IP address
1.“SSID Name”: Use the left or right arrow keys to select
   “Search”, pressand wait for the system to search for
   the wireless network, and then select the wireless network
   you want in the pop-up dialogue and press.
2.Select “Password”, pressto enter the soft keyboard,
  and then pressto exit from the soft keyboard after
  using the direction keys to input the password of the
  wireless network (no need to fill in if no password is set).
(Remarks:Supporting input method of multi-languages, press
 <Input Method> button on the TVpad remote to switch.)
3. “Network Type”: Use the left or right arrow keys to select
   “Automatically get IP address”, select “ok” and
   press，the system prompts “The wireless network
   succeeded”, then press to exit the program.
