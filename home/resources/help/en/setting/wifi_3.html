Description
Set the parameters of the wireless network. This device can
support the wireless encryption mode of WEP, WAP/WAP2,
and will automatically match the wireless encryption mode
when a valid password is filled in. As with “Network Settings”,
Wi-Fi Settings also supports the modes of manual and automatic.

Note
This device does not support wireless network and wired
network at the same time, but the system default wired
network when the two kinds of network mode accessed. We
suggest using wired network for better visual experience.
