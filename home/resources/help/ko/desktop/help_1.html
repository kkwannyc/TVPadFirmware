사용 절차
1.리모콘의 방향 버튼을 이용하여 응용프로그램을 선택한
  후 버튼을 클릭하여 관련 응용프로그램을 사용합니다.
2.임의 기능 구역에서 리모콘의 을 클릭하면 즉시 메인
   메뉴 화면으로 돌아 갑니다.

화면 설정
1.좌측 기능란의 위아래 순서는：응용 관리，고객 센터，
  설정.
2.우측의 응용 프로그램 단축키 리스트에는 고객님께서  
   다운로드한 응용프로그램을 보여 드립니다.
3.우하측구역은 본기기 상태를 표시합니다（선후 순위를
   구분하지 않음）：무선 네트워크 （Wi-Fi）, 유선 네트
   워크, USB 외장형 저장 장치， TF 카드，온라인 상태.

상세 정보:

유선 네트워크의 접속이 정상임을 표시 합니다.
