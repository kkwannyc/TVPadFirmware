사용 절차
1. 외장형 저장 장치 설비(USB메모리, TF카드, SD카드，
   외장하드 디스크 등)를 Tvpad 와 연결 후 상태 표시
   줄에서 표시되면 외장형USB 설비를 성공적으로 식별
   하였음을 표시 합니다.
2.방향 버튼을 이용하여 외장형 저장 장치의 도표를 선
   택하고 를 누르십시오.
3.팝업된 대화창에서 방향 버튼을 이용하여 “확인”을 선
   택한후 를 누르십시오.
4.시스템 포맷을 기다리며 “포맷 성공”창이 뜨면
  를 눌러 프로그램에서 나갑니다.

설명
외장형 USB메모리 설비를 EXT3방식으로 포맷시켜 응
용프로그램의 정상적인 플레이 운영을 만족하며 (단일
파티션인 외장형 설비만 지원하며 USB메모리에서 서
류함을 설정하지 않았음을 확인함) 설비를 포맷한후 시
스템의 서류 형식은 EXT3 디폴트로 됩니다. EXT3은 일
종의 저널링 파일 형식이며 EXT2 시스템의 확장형입니다.
