사용 절차
아래 작업 절차는 전부 TVpad 리모콘을 사용하여 방향
버튼으로 기능을 선택하고 버튼을 눌러 진행 합니다.

기능 리스트：
네트워크 검사：현재 연결된 네트워크의 상태 이상
                          여부를 검사하며 이상이 있을 경우
                          검사하고 네트워크를 재설정 합니다.
네트워크 설정：TVpad에서 정확한 유선네트워
                          크의 각 파라미터를 설정 합니다.
Wi-Fi       설정：TVpad에서 정확한 무선 네트
                          워크의 각파라미터를 설정 합니다.
언어        설정：Tvpad에서 여러가지 언어 버전
                           의 시스템환경을 제공 합니다.
시간대    설정： TVpad에서 시간대 설정을 진행합니다.
비디오    설정：비디오 출력 형식과 방식을 교체 합
                           니다.
포             맷：저장 장치 설비를 포맷 합니다.
