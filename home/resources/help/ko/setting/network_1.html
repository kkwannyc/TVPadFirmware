사용 절차
자동할당
1.“네트워크 형식”：좌우 방향 버튼을 이용하여 “자동할당”
   을 선택하고 “확인”을 선택한후 를 누르십시오.
2.시스템은 자동으로 네트워크 파라미터를 얻으며 “유선네
   트워크 설정 성공”을 제시한후 를 누르면 프로그램에
   서 나갑니다.

설명
유선 네트워크의 여러가지 파라미터 설정에는 수동/자동/
다이얼 PPPoE）등 3가지 방식이 있으며 일반 사용자는
“자동할당”방식을 권장합니다.（공유기가 성공적으로 설정
된 상황에서）고급 사용자는 수동 설정 또는 다이얼 설정을
선택하여 여러가지 파라미터를 설정할수 있습니다.

수동 설정
1.“ 네트워크 형식”：좌우 방향 버튼을 이용하여 “수동설정”
  을 선택하세요.
