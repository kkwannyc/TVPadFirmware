사용 절차
자동 할당
1.“SSID 이름”： 좌우 방향 버튼을 이용하여 “ 검색”을 선
   택하고 를 누른후 시스템의 무선 네트워크 검색을 기
   다리십시오. 대화창이 나타난후 사용할 무선 네트워크
   를 선택하고 를 누르십시오.
2.“패스워드”를 선택하고 를 누릅니다. 방향 버튼을 이
   용하여 무선 네트워크의 비밀 번호（비밀번호를 설정
   하지 않은경우 이부분을 입력할필요가 없음）를 입력
   한후 를 누르면 소프트 키보드에서 나갑니다.
  （비고: 소프트웨어 키보드의 언입력방식을 교체하려면
    리모콘상의 <입력모드> 버튼을 눌러 교체함.）
3. “ 네트워크 방식”： 좌우 방향 버튼을 이용하여 “ 자동
   할당”에서 “확인”을 선택한 후 를 누르고 “무선네트
   워크 설정 성공”을 제시하면 를 눌러 프로그램에서
   나갑니다.
