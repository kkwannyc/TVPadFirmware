작업설명
1.리모콘의 방향 버튼을 이용하여 선택한후 리모콘상의
  를 눌러주세요.

기능 옵션
충전, 충전기록, 정보, 유지보수 활성화.

충전
1.“충전”을 선택한후를 클릭하여 등록합니다.
2.방향버튼을 이용하여  카드번호,비밀번호를 순서대로
  선택하여 클릭 후 키보드에서 입력합니다. 설정
  완료후 를 클릭하여 소프트키보드에서 나갑니다.
 （주의: 다국어의 소프트키보드 입력방식을 제공하므로
   리모콘을 이용하여 <입력방식>버튼으로 필요하신 언어
   로 교체하실수 있습니다.）
3.모든수치 설정완료 후  방향버튼을 이용하여 “확인”을 선
   택한후 를 클릭하여 시스템에서 충전성공이 제시되
   면 을 클릭하여 시스템에서 나갑니다.
